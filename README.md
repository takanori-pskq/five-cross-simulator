# FiveCross Simulator

## Overview
Simulator for the playing card game [FiveCross](https://gist.github.com/takanori-pskq/efa6318f9383cf442d28f140d9adf7d5)

## Requirements
This code requires `cmake` to compile.

## Usage
To compile:
```
$ cmake .
$ make
```
To run:
```
./fc
```
